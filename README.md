# 去踏马的PDF密码

## 介绍

早期给一些 PDF 文件设置了**编辑密码**，时间一久密码遗忘。市面上的解锁工具要么强制会员、要么限制文件大小或次数，使用成本和不确定性都比较高，无法满足实际需求。

基于此，开发了这个 **本地离线的小工具**，用于**移除自己拥有的 PDF 文件的编辑密码**，无需联网、无需付费，操作简单直接。

目前仅支持 **Windows 平台**（作者当前使用环境为 Windows）。

------

### 功能特性

- 支持批量处理 PDF 文件
- 本地离线运行，不上传文件
- 无文件大小限制
- 操作简单，图形界面
- 仅尝试移除 **空密码 / 已知可打开的 PDF 编辑限制**

------

### 使用说明

1. 选择需要解锁的 PDF 文件（支持多选）
2. 选择输出目录（请勿与源文件同目录）
3. 点击“解锁 PDF 文件”
4. 解锁完成后将在目标目录生成新文件

------

### 使用声明

> 本工具仅用于处理**您本人合法拥有的 PDF 文件**。
>  请勿将其用于任何侵犯他人版权或违法用途，相关后果由使用者自行承担。

------
